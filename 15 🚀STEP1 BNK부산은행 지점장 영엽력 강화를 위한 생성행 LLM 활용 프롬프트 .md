## 🎯 핵심 액션 아이템 — 이 가이드의 배경

이 프롬프트 가이드는 아래 **7대 전략 과제**를 실행하기 위한 현장 도구로 제작되었습니다.

| 우선순위 | 액션 아이템 | 기대 효과 |
|:--------:|------------|----------|
| **1** | 본부 행정 부담 경감 + AI 업무도구 도입 | 지점장 영업 가용 시간 극대화 |
| **2** | 디지털 영업 인프라(태블릿뱅킹·예약 시스템) 전점 확대 | 영업 효율성·고객 경험 동시 향상 |
| **3** | 고객 데이터 기반 CRM 고도화 + 전담 고객제도 정착 | 맞춤 영업 → 교차판매율·고객 로열티 제고 |
| **4** | KPI 체계를 고객 장기가치 중심으로 재설계 | 단기 실적 압박 해소, 건전한 영업 유도 |
| **5** | 지점장 대상 AI·디지털·컨설팅세일즈·해양금융 교육 체계 강화 | 전문성 기반 차별화 영업력 확보 |
| **6** | 지역 특화 금융(해양·조선·소상공인) 지점장 전문화 | 부산은행만의 독보적 영업 경쟁력 구축 |
| **7** | 자율적 의사결정 문화 + 책임경영 체계 정착 | 현장 대응력·실행 속도 향상 |

## 1. 본부 지원 체계 혁신 — 행정 부담 경감

### 1-1. 보고서·기안 자동 생성

```
당신은 BNK부산은행 영업점 업무 지원 전문가입니다.

아래 데이터를 바탕으로 [○○지점]의 2026년 1분기 영업실적 보고서를 작성해 주세요.

[입력 데이터]
- 수신 잔액: 1,250억 원 (전분기 대비 +3.2%)
- 여신 잔액: 980억 원 (전분기 대비 +1.8%)
- 신규 고객 수: 342명 (개인 287명, 기업 55명)
- 교차판매율: 2.3건/고객
- 연체율: 0.42%
- 주요 신규 거래처: HJ중공업 2차 협력사 3곳, 양산 스마트물류단지 입주기업 5곳

[작성 조건]
1. 경영진 보고용 양식(요약 → 실적 상세 → 이슈 → 하반기 전략 순서)
2. 전분기 대비 증감 분석과 원인을 포함
3. 지역경제 동향(해양·조선·물류)과 연계한 시사점 기술
4. 하반기 중점 추진 과제 3가지를 구체적 실행 방안과 함께 제시
5. 분량은 A4 2페이지 이내
```

> 💡 **기대 효과** : 지점장이 보고서 작성에 소요하던 시간(평균 2~3시간)을 30분 이내로 단축하여 고객 접점 활동 시간을 확보할 수 있습니다. 데이터만 입력하면 일관된 품질의 보고서가 산출되므로 본부 검토 과정도 간소화됩니다.

### 1-2. 회의록·지시사항 자동 정리

```
아래는 BNK부산은행 [○○지점] 주간 영업회의 녹취 요약본입니다.
이 내용을 기반으로 다음을 정리해 주세요.

[녹취 요약]
"""
(여기에 회의 내용 텍스트 붙여넣기)
"""

[출력 형식]
1. 핵심 논의사항 (3줄 이내 요약)
2. 지점장 지시사항 (담당자·기한·구체적 액션 아이템 표 형태)
3. 미결 이슈 및 차주 확인 필요 사항
4. 본부 요청/에스컬레이션이 필요한 항목

표 형식으로 정리하고, 각 액션 아이템에 우선순위(상/중/하)를 부여해 주세요.
```

> 💡 **기대 효과** : 회의 후 후속 조치 관리가 체계화되고, 지점 내 커뮤니케이션 효율이 높아집니다. 지점장이 구두 지시한 내용이 문서화되어 실행 누락을 방지할 수 있습니다.

### 1-3. 규정·지침 즉시 검색 및 해석

```
당신은 BNK부산은행의 여신 규정 전문가입니다.

아래 상황에서 적용 가능한 여신 규정과 심사 기준을 안내해 주세요.

[상황]
- 고객: 양산 소재 조선 기자재 제조업체 (매출 50억, 업력 7년)
- 요청: 운전자금 대출 5억 원
- 특이사항: 최근 HJ중공업 신규 납품 계약 체결,
  기존 타행 대출 3억 원 보유

[요청 사항]
1. 적용 가능한 부산은행 여신 상품을 우선순위별로 추천
2. 각 상품의 심사 필요 서류 리스트
3. 생산적 금융 지원 대상 해당 여부 및 우대 조건
4. 심사 시 유의할 리스크 포인트
5. 예상 처리 소요 기간

실무자가 바로 활용할 수 있도록 체크리스트 형태로 정리해 주세요.
```

> 💡 **기대 효과** : 지점장과 영업 담당자가 복잡한 규정집을 뒤지는 시간을 절약하고, 고객 상담 현장에서 즉각적으로 정확한 정보를 제공할 수 있어 상담 전환율이 향상됩니다.

## 2. 디지털 무장 현장 영업 — 고객 상담 고도화

### 2-1. 고객 맞춤형 상담 시나리오 생성

```
당신은 BNK부산은행의 VIP 자산관리 전문 컨설턴트입니다.

아래 고객 프로필을 기반으로 맞춤형 상담 시나리오를 작성해 주세요.

[고객 프로필]
- 연령: 58세 / 자영업 (부산 해운대 음식점 2개 운영)
- 수신 잔액: 3.2억 원 (정기예금 위주)
- 여신: 사업자 대출 1.5억 원 (금리 4.8%)
- 월 평균 카드 매출: 8,000만 원
- 관심사: 은퇴 준비, 자녀(대학생) 학자금
- 최근 행동: 타행 ISA 상품 문의 이력

[작성 조건]
1. 상담 오프닝 멘트 (라포 형성)
2. 고객 니즈 파악을 위한 질문 리스트 (5개)
3. 추천 상품 포트폴리오 (예금·펀드·보험·대출 리밸런싱 포함)
4. 각 추천 상품의 고객 관점 혜택을 쉬운 언어로 설명
5. 예상 반론(타행 금리 비교 등)에 대한 대응 화법
6. 클로징 멘트 및 후속 조치 스케줄
```

> 💡 **기대 효과** : 지점장이 VIP 고객 상담 전에 체계적인 준비를 신속하게 마칠 수 있습니다. 고객 이탈 방지와 교차판매율 향상에 직결되며, 주니어 직원의 상담 역량 교육 자료로도 활용 가능합니다.

### 2-2. 기업 고객 사전 분석 및 접근 전략

```
당신은 BNK부산은행의 기업금융 전략 분석가입니다.

아래 정보를 바탕으로 신규 기업 고객 발굴을 위한 접근 전략을 수립해 주세요.

[대상 기업 정보]
- 기업명: (주)○○마린테크
- 업종: 친환경 선박 부품 제조
- 소재지: 부산 강서구 미음산업단지
- 매출: 120억 원 (전년 대비 +25%)
- 종업원: 85명
- 최근 동향: 노르웨이 선사 수출 계약 체결, 신공장 증설 검토 중
- 현재 주거래 은행: 국민은행

[분석 및 전략 요청]
1. 해당 기업의 금융 니즈 예측 (운전자금, 시설자금, 외환, 보증 등)
2. 부산은행이 제공 가능한 차별화 솔루션
   (해양금융 특화, RG 발급 역량, 생산적 금융 지원 등)
3. 의사결정자 접근을 위한 3단계 영업 시나리오
4. 경쟁은행(국민은행) 대비 부산은행의 어필 포인트
5. 첫 미팅에서 사용할 핵심 제안 메시지 (30초 엘리베이터 피치)
6. 예상 거래 규모 및 수익 시뮬레이션
```

> 💡 **기대 효과** : 지점장이 신규 우량 기업 발굴 시 체계적인 사전 분석과 전략 수립에 드는 시간을 대폭 줄이면서도 전문성 있는 접근이 가능합니다. 특히 부산은행의 해양금융 강점을 효과적으로 어필하는 데 활용할 수 있습니다.

### 2-3. 상품 비교 분석 자료 즉석 생성

```
고객이 부산은행 정기예금과 타행(신한은행, 국민은행) 정기예금을 비교하고 있습니다.

아래 조건으로 고객 설명용 비교 자료를 만들어 주세요.

[조건]
- 예치 금액: 1억 원
- 예치 기간: 12개월
- 고객 유형: 부산은행 급여이체 고객 (우대금리 적용 대상)

[출력 형식]
1. 3개 은행 상품 비교표 (기본금리, 우대금리, 세후 실수령 이자)
2. 부산은행 선택 시 추가 혜택 (부수거래 우대, 동백전 연계 등)
3. 고객에게 설명할 때 사용할 핵심 문장 3개
4. "금리만 비교하면 비슷하지만, 종합적으로 부산은행이 유리한 이유"를
   논리적으로 정리

※ 현재 공시금리 기준으로 작성하되, 금리는 변동 가능하므로
  반드시 실제 확인이 필요하다는 안내를 포함해 주세요.
```

> 💡 **기대 효과** : 고객이 타행 이동을 고려할 때 현장에서 즉각적으로 설득력 있는 비교 자료를 제시하여 이탈을 방지할 수 있습니다. 지점장이 금리 외의 종합적 가치를 전달하는 데 효과적입니다.

## 3. 고객 세분화 기반 맞춤 영업 — CRM 고도화

### 3-1. 고객 세그먼트별 캠페인 기획

```
당신은 BNK부산은행의 CRM 마케팅 전략가입니다.

[○○지점]의 고객 데이터를 기반으로 세그먼트별 타겟 캠페인을 기획해 주세요.

[지점 고객 현황]
- 총 거래 고객: 12,500명
- 세그먼트 구분:
  A. 급여이체 직장인 (30~40대, 4,200명) — 평균 수신 2,800만 원
  B. 자영업자·소상공인 (40~50대, 2,100명) — 평균 여신 8,000만 원
  C. 은퇴·시니어 (60대 이상, 3,800명) — 평균 수신 5,500만 원
  D. 대학생·사회초년생 (20대, 1,500명) — 평균 수신 350만 원
  E. 기업 법인 (900개) — 평균 여신 3.2억 원

[각 세그먼트별 요청 사항]
1. 핵심 금융 니즈 분석
2. 추천 타겟 상품 (교차판매 기회)
3. 접촉 채널 및 타이밍 (방문·전화·문자·앱 푸시)
4. 상담 시 핵심 멘트 (세그먼트별 차별화)
5. 1개월 내 실행 가능한 미니 캠페인 아이디어
6. 예상 성과 지표 (KPI)
```

> 💡 **기대 효과** : 지점장이 전체 고객을 막연하게 관리하는 것이 아니라, 데이터 기반으로 세그먼트별 우선순위와 차별화된 접근법을 갖출 수 있습니다. 한정된 영업 인력을 가장 효과적으로 배분하는 의사결정에 도움이 됩니다.

---

### 3-2. 이탈 위험 고객 선제 대응 시나리오

```
아래는 [○○지점]에서 최근 3개월간 이탈 위험 신호가 감지된 고객 리스트입니다.

[이탈 위험 고객 패턴]
- 고객 A: 급여이체 해지 후 타행으로 변경, 정기예금 만기 도래 (2주 후)
- 고객 B: 카드 사용 급감 (월 300만→50만), 타행 앱 설치 이력
- 고객 C: 사업자 대출 상환 완료 후 6개월간 거래 없음
- 고객 D: VIP 고객, 최근 증권사 자산이동 (수신 5억→2억)

[각 고객별 요청]
1. 이탈 원인 추정 (가능한 시나리오 3가지)
2. 선제 접촉 타이밍 및 방법
3. 리텐션 상담 대화 스크립트 (자연스러운 톤으로)
4. 제안 가능한 우대 조건 또는 맞춤 솔루션
5. "하지 말아야 할 것" (역효과 우려 행동)
6. 상담 후 후속 관리 프로세스
```
